# Rustup
## Environment manager for rust
### Macos/Linux
```sh
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

### Windows
1. Go to [https://rustup.rs/](https://rustup.rs/)
1. Download and install rustup-init.exe
