# Std Types
## `String` vs `&str`
* `String`: Heap allocated growable buffer
* `&str`: Reference to fixed size buffer
## `Vec<T>` vs `[T; N]`
* `Vec`: Growable array (ArrayList/List)
* `[T; N]`: Fixed size array
## `Box<T>`
* Heap allocated pointer
